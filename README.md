# Super Mario Bros. 1 Clone (Educational Project)

> ⚠️ **Important Notice**
>
> This project was created solely for educational, learning, and research purposes. It is not intended for commercial use and does not aim to replace, compete with, or redistribute the original game.

## Purpose

The purpose of this project is to study and learn fundamental game development concepts, including:

- Sprite rendering
- Physics and collision detection
- Character movement
- Camera systems
- State management
- Level design and progression
- User input handling (keyboard, controller, etc.)
- Game architecture and code organization

The decision to recreate mechanics inspired by a classic game was made exclusively because of its educational value and historical importance in game development.
